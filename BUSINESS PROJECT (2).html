Problem Statement

The company has introduced a new landing page and wants to evaluate whether it performs better than the old page. Specifically, the goal is to determine if the new page increases user engagement (time spent on the page) and improves conversion rates, while also checking if language preference influences these outcomes.

Objectives

  1. Explore and clean the dataset using pandas.
  2. Visualize distributions and group comparisons with matplotlib and seaborn.
  3. Perform hypothesis tests:
    • t-test: Compare mean time spent between old and new pages.
    • Proportion test: Compare conversion rates between old and new pages.
    • Chi-square test: Assess independence between conversion and language preference.
    • ANOVA: Test differences in time spent across languages (new page only).
  4. Interpret results and provide business insights.

# Step 6: Hypothesis Testing

# 1. Time spent on new vs old page
old = df[df['landing_page']=="old"]["time_spent_on_the_page"]
new = df[df['landing_page']=="new"]["time_spent_on_the_page"]

t_stat, p_val = stats.ttest_ind(new, old, alternative="greater")
print("T-test: T-statistic =", t_stat, "P-value =", p_val)

T-test: T-statistic = 3.7867702694199856 P-value = 0.0001316123528095005

Observation: Time Spent on New vs Old Page

  • The p-value is < than 0.05 ,which shows that the users spend significantly more time on the new landing page.

# 2. Conversion rate comparison
convert_counts = np.array([
    (df[(df['landing_page']=="new") & (df['converted']=="yes")].shape[0]),
    (df[(df['landing_page']=="old") & (df['converted']=="yes")].shape[0])
])
nobs = np.array([
    df[df['landing_page']=="new"].shape[0],
    df[df['landing_page']=="old"].shape[0]
])

z_stat, p_val = proportions_ztest(convert_counts, nobs, alternative="larger")
print("Z-test: Z-statistic =", z_stat, "P-value =", p_val)

Z-test: Z-statistic = 2.4077170617153842 P-value = 0.008026308204056278

Observation: Conversion Rate Comparison

  • The p-value is < than 0.05, which suggests that conversion rate is significantly higher on the new page.

# 3. Conversion vs language independence
contingency = pd.crosstab(df['converted'], df['language_preferred'])
chi2, p_val, dof, expected = stats.chi2_contingency(contingency)
print("Chi-square: Chi2 =", chi2, "P-value =", p_val)

Chi-square: Chi2 = 3.0930306905370837 P-value = 0.21298887487543447

Observation: Conversion vs Language Independence

  • The p-value is > than 0.05, showing that conversion outcomes are not affected by language preference.

# 4. Time spent across languages (new page only)
new_page = df[df['landing_page']=="new"]
groups = [group["time_spent_on_the_page"].values for name, group in new_page.groupby("language_preferred")]
f_stat, p_val = stats.f_oneway(*groups)
print("ANOVA: F-statistic =", f_stat, "P-value =", p_val)

ANOVA: F-statistic = 0.8543992770006822 P-value = 0.43204138694325955

Observation: Time Spent Across Languages (New Page)

  • The p-value is > than 0.05, indicating that time spent is consistent across languages.
